Frequently Asked Questions

Is my data secure when using LocalDrive?

Yes. LocalDrive only makes your files accessible on your local network. It doesn't expose them to the internet. The application doesn't send any data to external servers.

Can I access my files from outside my home network?

LocalDrive is designed for local network use only. If you need remote access, you would need to set up additional networking solutions like a VPN to your home network.

Is there a mobile app version?

Currently, LocalDrive works through a web interface that's fully responsive and works well on mobile browsers. A dedicated mobile app might be developed in the future.

How does the QR code connectivity work?

When you start the LocalDrive server, it generates a QR code containing the local network URL. Simply scan this with your mobile device's camera to open the interface in your mobile browser.

What makes LocalDrive different from other file sharing apps?

LocalDrive combines spiritual inspiration with modern technology, featuring an elegant Krishna-themed design. It's also completely open-source, respects your privacy, and includes unique features like the Windows context menu integration and built-in media streaming.
