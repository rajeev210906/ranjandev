Semester IV Syllabus

Bachelor of Technology in Computer Science & Engineering - Data Science

Design Thinking for Entrepreneurship and Startups (HSMCD401)

Credits: 3 | Contact Hours: 45

Module I: Introduction to Design Thinking and Entrepreneurship

Definition, Need, Stages of Design Thinking Process, and Business Structures.

Module II: Creative Thinking

Understanding Problem Solving and Testing Creative Problem Solving.

Module III: Process of Product Design

Engineering Product Design, Design Thinking Approach, and Assignment Examples.

Module IV: Prototyping & Testing

Rapid Prototype Development Process, Testing, and Group Marketing.

Module V: Design Thinking & Customer Centricity

Enhancing Customer Experience, Parameters of Product Design.

Module VI: Business Ideas and Implementation

Discovering Ideas, Business Plans, and Activity Mapping.

Module VII: Managing the Business

Market Analysis, Financing Startups, and Talent Recruitment.

Module VIII: Intellectual Property Rights

IPR Concepts, Marketing Use, and International Context.

Operating Systems (PCC-CSD401)

Credits: 3 | Contact Hours: 45

Module I: Introduction to Operating Systems

Roles, Services, System Calls, and Types of Operating Systems.

Module II: Process Management

Process Scheduling, Threads, and Real-Time Scheduling Algorithms.

Module III: Process Synchronization

Critical Section Problem, Semaphores, and Classical IPC Problems.

Module IV: Deadlocks

Deadlock Prevention, Detection, and Banker’s Algorithm.

Module V: Memory Management

Paging, Segmentation, and Virtual Memory Concepts.

Module VI: File and I/O Management

File Structures, Disk Scheduling, and Device Management.

Design and Analysis of Algorithms (PCC-CSD402)

Credits: 3 | Contact Hours: 45

Module I: Algorithm Basics

Stages of Algorithm Development and Recurrence Relations.

Module II: Algorithm Design Techniques

Divide and Conquer, Greedy Techniques, and Dynamic Programming.

Module III: String Matching Algorithms

Naive Algorithm, KMP Algorithm, and Rabin-Karp Algorithm.

Module IV: Graph Algorithms

Shortest Path Algorithms, Network Flows, and Cycle Canceling Algorithm.

Module V: Complexity Classes

P, NP, NP-Complete, and Reducibility.

Module VI: Approximation Algorithms

Set Cover, Vertex Cover, and Randomized Algorithms.

Object Oriented Programming (PCC-CSD403)

Credits: 3 | Contact Hours: 45

Module I: Object Oriented Concepts

Classes, Objects, Inheritance, Polymorphism, and Encapsulation.

Module II: Java Basics

Java Syntax, Operators, Control Statements, Arrays, and String Handling.

Module III: Advanced Java Features

Inheritance, Interfaces, Packages, and Abstract Classes.

Module IV: Exception Handling and Multithreading

Try-Catch, User-Defined Exceptions, Threads, and Synchronization.

Module V: GUI Programming

AWT, Swing, and Event Handling in Java.

Formal Language & Automata Theory (PCC-CSD404)

Credits: 3 | Contact Hours: 45

Module I: Introduction

Alphabets, Languages, Grammars, and Chomsky Hierarchy.

Module II: Regular Languages

Regular Expressions, DFA, NFA, and Properties of Regular Languages.

Module III: Context-Free Languages

CFG, PDA, Parse Trees, and Ambiguity in CFG.

Module IV: Context-Sensitive Languages

Context-Sensitive Grammars and Linear Bounded Automata.

Module V: Turing Machines

Turing Machine Basics, Recursive Languages, and Decidability.

Discrete Mathematics (PCC-CSD405)

Credits: 4 | Contact Hours: 60

Module I: Sets, Relations, and Functions

Set Operations, Cartesian Products, Binary Relations, and Functions.

Module II: Counting Techniques

Permutations, Combinations, Generating Functions, and Recurrence Relations.

Module III: Propositional Logic

Syntax, Semantics, Logical Equivalence, and Quantifiers.

Module IV: Algebraic Structures

Groups, Rings, Fields, and Boolean Algebra.

Module V: Graphs and Trees

Graph Properties, Isomorphism, Eulerian Paths, and Spanning Trees.

Operating Systems Lab (PCC-CSD491)

Credits: 1.5 | Contact Hours: 45

Module I: Process Management

Starting Processes, Context Switching, and Process States.

Module II: Scheduling

Implementing FCFS, SJF, and Round Robin Scheduling Algorithms.

Module III: Semaphore Programming

Semaphores, Pipes, and IPC Mechanisms.

Module IV: Deadlocks

Banker’s Algorithm, Detection, and Recovery.

Module V: Shell Scripting

Basic Shell Commands, Scripts, and Control Structures.

Design and Analysis of Algorithms Lab (PCC-CSD492)

Credits: 1.5 | Contact Hours: 45

Module I: Sorting Algorithms

Implementing Quick Sort and Merge Sort.

Module II: Graph Algorithms

BFS, DFS, and Minimum Spanning Trees (Kruskal’s and Prim’s).

Module III: Dynamic Programming

Knapsack Problem and Floyd’s Algorithm.

Module IV: Backtracking

N-Queens Problem and Graph Coloring.

Module V: Approximation Algorithms

Traveling Salesperson Problem.

Object Oriented Programming Lab (PCC-CSD493)

Credits: 1.5 | Contact Hours: 45

Module I: Java Basics

Simple Java Programs, Classes, and Objects.

Module II: Advanced Java Features

Inheritance, Interfaces, and Packages.

Module III: Exception Handling

Try-Catch Blocks, User-Defined Exceptions.

Module IV: Multithreading

Thread Lifecycle, Synchronization, and Deadlocks.

Module V: GUI Development

AWT and Swing Components.
